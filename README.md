# sentry/quant — A股量化选股系统

**目的**：自动扫描全市场 A 股（~11000只），识别符合历史强势形态的起爆候选股，叠加科创/创业板平台突破策略、事件驱动（公告/新闻）分析，每个交易日晚 8 点通过企业微信推送可操作的选股建议，并支持双向对话查询。

---

## 每日工作流

```
14:55  A股收盘
  ↓
15:xx  手动：复制 hsjday{mmdd}/ 到 Windows ~/Downloads/YYYYMMDD/
  ↓
20:00  [cron] q-pick-today-batch      形态选股 + q-fin深度分析 + 推送
20:15  [cron] q-news-daily-batch      重大公告 + LLM利多/利空分析 + 推送
20:30  [cron] q-kechuang-batch        科创/创业板平台突破扫描 + 推送 Top3
20:45  [cron] q-ipo-watch             近7天新股列表写入缓存
  ↓
随时   企业微信「福宝抓股」应用双向对话（q-chatbot 常驻）
随时   http://localhost:8501  本地 dashboard 4板块
```

## 系统架构

```
┌──────────────────────────────────────────────────────────┐
│                       数 据 层                            │
│  TDX 日线(hsjday)  TDX 财报(tdxfin)  cninfo公告  新闻RSS  │
└─────────────────────────┬────────────────────────────────┘
                          │
┌─────────────────────────▼────────────────────────────────┐
│                       分 析 层                            │
│                                                          │
│  q-seed ────── KNN 形态相似度 (6模板)                     │
│  kechuang ──── 平台突破规则引擎 (科创+创业板)              │
│  q-backtest ── walk-forward 回测                          │
│  q-fin ─────── 深度事件调研 (DeepSeek + DDG/Tavily)       │
│  q-news ─────── 公告/新闻分析 (LLM 利多/利空推理链)        │
│  q-ipo-watch ── 近7天新股扫描                             │
└─────────────────────────┬────────────────────────────────┘
                          │
┌─────────────────────────▼────────────────────────────────┐
│                       输 出 层                            │
│  q-push → 企业微信群推送                                  │
│  q-chatbot → 企微应用双向对话                             │
│  q-dashboard → Streamlit 单页4板块 :8501                  │
└──────────────────────────────────────────────────────────┘
```

## 命令一览

| 命令 | 职责 | 触发 |
|------|------|------|
| `q-pick-today` | 形态信号+q-fin+推送（主流程） | cron 20:00 |
| `q-news` | cninfo公告+新闻+LLM分析+推送 | cron 20:15 |
| `q-kechuang-batch` | 科创/创业板平台突破扫描+推送 | cron 20:30 |
| `q-ipo-watch` | 近7天新股列表生成 | cron 20:45 |
| `q-chatbot-start` | 企微双向对话服务（Flask+cloudflare） | 常驻 |
| `q-seed-hongjing` | 宏景型形态扫描 | 被 q-pick-today 调用 |
| `q-backtest-fast` | walk-forward 回测 | 手动 |
| `q-fin` | 深度基本面+事件调研 | 被各批处理调用 |
| `q-push` | 企业微信 webhook 推送 | 被各批处理调用 |
| `q-sync` | 同步 TDX 日线数据 | 手动/批处理 |
| `q-sync-fin` | 同步 TDX 财报数据 | 手动（季报期） |
| `q-dashboard` | Streamlit 本地可视化 | 常驻 :8501 |
| `q-clean` | 清理过期日志/缓存 | 手动 |

## 科创/创业板平台突破策略

- 针对 688xxx/300xxx/301xxx，上市1~5年内的新股
- 条件：低振幅横盘（振幅<30%） + 放量突破（量比>1.5x）+ 非下跌趋势
- 今日结果：11只命中，q-fin评选Top3推送
- 验证案例：品高股份(+4.3%后启动)、寒武纪(+14%后上涨数百%)

## 企业微信双向对话（q-chatbot）

在「福宝抓股」应用发送：

| 输入 | 响应 |
|------|------|
| `分析 688211` | q-fin深度分析+LLM评论员报告（约3-5分钟） |
| `推荐几只股票` | 当日形态选股缓存（秒回）或触发扫描 |
| `今天有什么公告` | 当日公告热点缓存（秒回） |
| `科创板` | 科创突破策略缓存（秒回） |
| 其他问题 | DeepSeek A股问答 |

> 缓存当日有效，第二天自动清除重新扫描

## Dashboard（http://localhost:8501）

单页4板块布局，手机访问需通过 cloudflare tunnel：

- **🌱 形态选股** — q-seed推荐 + q-fin详情，含股东研究链
- **📰 公告热点** — q-news今日题材，含cninfo原始链接
- **🚀 科创突破** — 科创/创业板平台突破Top3
- **🆕 近期新股** — 近7天上市新股，发行价/现价/涨跌
