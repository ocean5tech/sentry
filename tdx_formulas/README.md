## 核心概念

### 起爆日 (Launch Day)
满足以下条件的交易日：
- 涨幅 ≥ 5%
- 成交量 ≥ 20日均量的2倍

### Wave形态
- 至少2个起爆日
- 波峰递增（HH: Higher High）
- 波谷递增（HL: Higher Low）
- Pierce条件：回调不破前期起爆点

### 反喇叭口收敛
- 27日振幅 > 18日振幅 > 9日振幅
- 表示价格波动逐渐收窄
- 通常预示即将突破

## 使用方法

### 在通达信中导入公式

1. 打开通达信软件
2. 功能 → 公式系统 → 公式管理器
3. 选择"条件选股公式"
4. 新建公式，复制对应策略的内容
5. 保存并执行选股

### 选股建议

1. **单策略选股**: 使用单个策略文件，精准匹配特定形态
2. **综合选股**: 使用综合策略，覆盖更多机会
3. **交叉验证**: 多个策略同时选中的股票，可靠性更高

### 买入时机

- **信号日**: 出现"★★买点★★"标记的当天
- **实际买入**: T+1日开盘买入
- **止损**: 跌破起爆日开盘价
